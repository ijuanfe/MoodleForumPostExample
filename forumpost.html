This is a simple template to embed code in HTML tables of size 1x1 with black background in a Moodle forum post.

Every HTML tag must be inside a div tag. Some examples:

  • <div> </br> </div>

  • <div> <p> .. </p> </div>

Table example:

%%%%% Oz programming language example %%%%%

fun {SolveAll F}
   L = {Solve F}
   proc {TouchAll L}
      if L == nil then skip else {TouchAll L.2} end
   end
in
   {TouchAll L}
   L
end

End of the forum post.
